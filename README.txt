Note:
While executing the functions that run the game (e.g. runMatch, runGame) you need to put first the OnitamaAgent 
who has the RedPlayer and, in second place, the one who has the BluePlayer. Otherwise, the program will throw an exception.
Ej: (RandomAgent RedPlayer, RandomAgent BluePlayer) --> Correct input.
    (RandomAgent BluePlayer, RandomAgent RedPlayer) --> Incorrect input.
